# W1.8 — Integration recipes for agent devs

Part of the [pivot plan](README.md). Workstream 1. The actual customer-facing copy-paste integrations for the wedge audience (agent dev platforms).

**Workstream:** 1
**Dependencies:** W1.1 (so the keys recipes reference are real)
**Status:** planned

## Acceptance criteria

- `examples/langchain/` — minimal Python agent that calls `verify_wallet` via MCP, ≤30 lines, runnable.
- `examples/crewai/` — same shape for CrewAI.
- `examples/claude-code/` — `.mcp.json` config + a one-liner showing how to use the tool inside Claude Code.
- Each example has a README with copy-pasteable setup steps.

## Files

- `examples/langchain/{agent.py,README.md}`
- `examples/crewai/{agent.py,README.md}`
- `examples/claude-code/{README.md,.mcp.json}`

## Test spec
