# docs/archive/

Hackathon-era artifacts, kept for history. None of this is load-bearing for the
current product — it's preserved so the why-behind-the-shape isn't lost.

- `WARD-o.html`, `WARD-o-business.html`, `WARD-o.pptx` — the original pitch
  decks (technical + business). The product framing has since shifted to a free,
  honest tool; the live copy lives on the site and in the top-level README.
- `research/` — exploratory notes that predate or sit outside a single feature.
- `real-wallet-tests/` — fixture wallets, per-run reports, and side-by-side
  diffs. Still the source-of-truth provenance for `src/fixtures/wallets.ts` (the
  regression anchor), so it's kept verbatim.
